<h2>Managing User Accounts in Active Directory</h2>

<p>
Active Directory allows IT administrators to manage user accounts, including enabling or disabling accounts and resetting user passwords.
</p>

<h3>1. Open Active Directory Users and Computers</h3>

<p>
Open <b>Active Directory Users and Computers</b>, then select your organization's domain from the left navigation pane.
</p>

<p>
  <img width="745" height="964" alt="image" src="https://github.com/user-attachments/assets/6dbde3cb-8f37-485f-b1e7-2728e4740c8d" />

</p>


<h3>2. Open the Search Window</h3>

<p>
Click the <b>Search</b> icon (magnifying glass) located in the top-right corner of the window, highlighted in yellow.
</p>
<p>
<img width="745" height="964" alt="image" src="https://github.com/user-attachments/assets/9de00226-06f3-4526-b38f-887a3055eae8" />
</p>


<h3>3. Search for the User</h3>

<p>
When the search window pops up, select <b>Users, Contacts, and Groups</b>. Enter the user's name in the <b>Name</b> field, then click <b>Find Now</b>.
</p>
<p>

  <img width="1055" height="626" alt="image" src="https://github.com/user-attachments/assets/09ffb8bd-d138-4470-81c9-026982441304" />

</p>




<h3>4. Manage the User Account</h3>

<p>
After locating the user in the search results, right-click user's name to display the available account management options, then you can perform common account management tasks such as:
</p>
<p>
  <img width="516" height="592" alt="image" src="https://github.com/user-attachments/assets/45d7766e-f0e9-447e-ae5b-21eab5417189" />

</p>
<ul>
<li>Enable a user account.</li>
<li>Disable a user account.</li>
<li>Reset the user's password.</li>
</ul>
